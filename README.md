# Gideon Finance

A modern, dark-themed financial control system.

## Features

- **Modern Dark Design:** A sleek and modern dark theme.
- **Dashboard:** A comprehensive dashboard with a balance chart and a list of recent transactions.
- **Responsive Layout:** The application is designed to be responsive and work on different screen sizes.

## Stack

- HTML5
- CSS3
- JavaScript
- Chart.js
- Google Fonts (Inter)
